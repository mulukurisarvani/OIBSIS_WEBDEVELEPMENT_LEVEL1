
# 🌐 Task 2 – Personal Portfolio Website

## 📌 Project Overview

This project is a **Personal Portfolio Website** created using **HTML and CSS**.
The purpose of this project is to showcase personal skills, projects, and contact information in a clean and professional layout.

It is a beginner-friendly web development project to practice webpage structure and styling.

---

## 🚀 Features

* Navigation bar
* Hero section with profile image
* About Me section
* Skills section
* Projects section
* Contact section
* Footer
* Clean dark theme design
* Responsive layout using Flexbox

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox (for layout design)
